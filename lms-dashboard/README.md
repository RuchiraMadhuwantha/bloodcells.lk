# LMS Dashboard

A comprehensive Learning Management System built with the MERN stack (MongoDB, Express, React, Node.js).

## Project Structure

- **client/**: React Frontend (Vite + TailwindCSS)
- **server/**: Node.js Backend (Express + MongoDB)

## Getting Started

### Prerequisites

- Node.js installed
- MongoDB running locally or a connection string

### Installation

If dependencies are not installed automatically:

1. **Client**:
   ```bash
   cd client
   npm install
   ```

2. **Server**:
   ```bash
   cd server
   npm install
   ```

### Running the App

1. **Start the Backend**:
   ```bash
   cd server
   npm run dev
   ```
   (Server runs on port 5000)

2. **Start the Frontend**:
   ```bash
   cd client
   npm run dev
   ```
   (Client usually runs on http://localhost:5173)

### Features

- **Double Slider Authentication**: Login and Register with animated panel (matches design).
- **Admin/Student Roles**: (Backend structure ready).
